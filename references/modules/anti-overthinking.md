# Anti-Overthinking Module

Use this module to keep marathon prompts productive instead of endless.

```text
ANTI-OVERTHINKING:
Long runtime is for evidence-backed progress, not indefinite polishing.

Do not continue if:
- all success criteria are verified
- the next action only polishes wording or style without changing evidence
- two quality passes show no material improvement
- risk or scope is increasing faster than confidence
- the next action expands the mission instead of completing it

If marginal evidence gain is low, stop with DONE if criteria are met, or PARTIAL DONE/BUDGET EXHAUSTED with a precise handoff if they are not.
```
